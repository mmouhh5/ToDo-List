# 📝 To-Do-List-App
- **🌐 Live Demo:** (https://todo-list-0o4j.onrender.com)
A lightweight web application built to practice the fundamentals of **CRUD** (Create, Read, Update, Delete) operations using Python and the Flask framework.

## 🚀 The "Why" behind this project
I built this project to understand how a server manages a "list" of data that changes based on user input. 

## 🛠️ Current Tech Stack
- **Language:** Python 3.x
- **Framework:** Flask (Backend)
- **Frontend:** HTML5 & CSS3
- **State Management:** Python Lists (Memory-based)

## 🚧 Work in Progress (Development Roadmap)
The project follows an iterative development cycle. I am currently learning how to bridge the gap between temporary memory and permanent storage.
- [x] **Phase 1:** Basic UI and "Add Task" POST request and "Delete Task" functionality.
- [x] **Phase 2:** Real-time list rendering using Jinja2 templates.
- [X] **Phase 3:** Integrate JSON File I/O for data persistence (Saving tasks).
- [ ] **Phase 4:** Add a sorting feature to sort tasks by date.
- [ ] **Phase 5:** Add a History feature to track completed tasks.
- [ ] **Phase 6:** UI/UX Polish: Add Dark Mode and task filtering to filter finished unfinished tasks.

## 💻 Running the Project
1. Clone the repository.
2. Ensure Flask is installed: `pip install flask`.
3. Run `python app.py` and navigate to `localhost:5000`.

**Note:** The live demo is hosted on a free tier and may take up to 60 seconds to wake up on the first load.
